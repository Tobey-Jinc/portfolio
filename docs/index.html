3DVis

C# Unity Project Management

3DVis is a scene building tool for the UniSC's CAVE2. Typically you would need to create a whole Unity app to get something on the CAVE. But 3DVis allows you to develop scenes on the CAVE allowing for fast iteration times. It also significantly lowers the barrier of entry for developing CAVE scenes.

I was the lead programmer and developer on this project and worked with 3 others who handled UI, modelling, management, and development of a companion app.
