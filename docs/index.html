3DVis

Developed for UniSC, 3DVis is a tool for the CAVE which is a cutting edge spatialisation environ-... it's a cool room with 80 or so screens lining the walls all around you. The CAVE is used for simulations. For instance, it could be used to train paramedics to perform actions in a stressful environment.

Usually you would need to make a whole Unity project to get something on the CAVE. 3DVis is an all in scene builder that runs on the CAVE, allowing for fast iteration.
